# MARKDOWN-CONVERTOR-TO

## Usage

**Markdown to javascript object converter**

```js
import { mdToObj } from 'mdTo.js';
let obj = mdToObj(mdText);
 ```

**Markdown to JSON converter**
```js
import { mdToObj } from 'mdTo.js';
let obj = mdToObj(mdText);
let json = JSON.stringify(obj, null, 2)
 ```
